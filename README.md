# 📅 Calculender - מחולל אירועים עבריים ליומן

**Calculender** היא אפליקציית Web חכמה המאפשרת יצירת קובץ יומן (`.ics`) עבור אירועים חוזרים לפי התאריך העברי (ימי הולדת, ימי נישואין, אזכרות וכו') ל-100 שנים קדימה. 

האפליקציה פותרת את אחד האתגרים הגדולים בניהול יומן דיגיטלי: סנכרון התאריך העברי (שתלוי בשקיעת החמה ומשתנה משנה לשנה) עם היומן הלועזי (Google Calendar, Outlook, Apple Calendar).

![Calculender Screenshot](https://raw.githubusercontent.com/hrsi56/Calculender/main/public/screenshot.png) *(מומלץ להוסיף צילום מסך של האפליקציה לכאן)*

## ✨ פיצ'רים מרכזיים

* **המרה חכמה מלועזי לעברי:** הזן תאריך לועזי, סמן אם האירוע התרחש לפני או אחרי השקיעה, והמערכת תחשב אוטומטית את התאריך העברי המדויק.
* **טיפול בשנים מעוברות (אדר א' / אדר ב'):** אלגוריתם חכם שיודע לזהות מתי אירוע צריך לחול באדר א', באדר ב', או באדר של שנה רגילה.
* **חישוב שקיעות אסטרונומי מדויק:** חישוב זמני השקיעה על דקה, תוך התחשבות בקואורדינטות הגיאוגרפיות **ובגובה הטופוגרפי (Elevation)** של ערים מרכזיות בישראל ובעולם.
* **תצוגת יומן כפולה ולא מציקה:** * אירוע יומי (All-Day Event) ליום העברי עצמו (שלא חוסם שעות ביומן).
  * אפשרות לאירוע "תזכורת" קצר של 15 דקות בדיוק בשעת השקיעה בערב שלפני.
* **ייצוא מהיר:** הורדת קובץ `.ics` בלחיצת כפתור, מוכן לייבוא לכל תוכנת יומן.

## 🛠️ טכנולוגיות (Tech Stack)

**Frontend:**
* React + TypeScript
* Vite
* עיצוב מותאם אישית (CSS/Variables) עם תמיכה מלאה ב-RTL.

**Backend:**
* Python + FastAPI
* [pyluach](https://pypi.org/project/pyluach/) - לחישוב והמרת תאריכי לוח עברי.
* [astral](https://pypi.org/project/astral/) - לחישובים אסטרונומיים של זמני זריחה ושקיעה.
* [ics](https://pypi.org/project/ics/) - ליצירת אובייקטי היומן.

**תשתית (DevOps):**
* Docker (Multi-stage build)
